RobotScratch -

RobotScratch is a Hexapod controlling app build on iOS platform. It is used to mainpluate the motions for the Hexapod. You can also used this app to control indiviual servo in the Hexapod. Health check, health report is also supported in this app. One of the most awesome funtionality is wireless firmware upgrade. No need to connect your robot with the PC, and download the firmware, you just need to make use of the app, and firware will automatically downloaded via Bluetooth. And you can also use RobotScratch to create a series motion control frame, and download to the robot so that the robot will move and dance according to your motion frames. It is really funny.

Features:

  • Individual servo control
  • Motion editor for the robot, motion frame download and play
  • Wireless Firmware upgrade, easy and stable
  • Servo degree adjustment
  • Minimum & Maximum degree setting for each servo
  • Live Health report for the connected Hexapod
  • Gamepad style control

Looking Inside: RobotScratch is build with the Bluetooth 4.0, aka BLE. Objective-C is the development language. I have also make use of the ReactiveCocoa framework to program in FRP pattern in some modules.

For the Hexapod control software, I have make use of the FreeRTOS to build a real-time system for montion control, BLE frame handling, health checking. And a set of BLE application frames are developed to help comunication between app and Hexapod.

For more detail about this project, you can download my master degree paper.

QuickBlue - QuickBlue is an app for developer to communicate with BLE 4.0 (Bluetooth Low Energy) peripherals.

Features:

  • Automatic data read/write with user define interval
  • Action recording and replay
  • Read/Write data save to file, and send via email.
  • Both HEX/ASCII mode supported

Looking Inside: Quickblue is built with Objective-C, and CoreBluetooth, and it also make heavy used of GCD in asynchornous way.

Easyfly - Easyfly is an LBS base flight service discovery platform. Easyfly is platform to connect flight service provider with potential customers. Easyfly is a B2C like platform, but what Easyfly offer is not a real product, but a flight service. For example, flight trail, aircraft exhibition, Aerial photography, flight trainning, etc. Through Easyfly, customers can make appointment to the service provider, and pay the charge online. Easyfly also offers aircraft database for the aircraft lovers, we have already included more than 6000+ aircraft in our database, and the number is still increasing.

Features:

  • Nearby flight service discover
  • Service provider information browse, including rating, how many services had been sold, etc.
  • Flight service detail browse, including comment from other customers, service rating, etc.
  • Map based service discover
  • Aircraft database inquiry, filter by criteria.
  • Comment on the service you have brought.
  • Pay with Wechat or Alipay online, make appointment online.

Looking Inside: Easyfly is sperated by backend, frontend and app. For the backend, it is platform build with Python & Django & REST Framework. For the frontend, it is build with Semantic-UI & JQuery. For the app, it is build with Swift 3.0 and various open source library. MVVM pattern is what I have used to develop this app.
